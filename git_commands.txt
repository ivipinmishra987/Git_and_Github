git init = Initialized an empty git vcs repository

git status= checks the status of the vcs

git add <filename> = untracked to staged(red colour to green colour)

git rm --cached <filename> = staged to untracked

git commit -m "" =  stages to tracked

git restore <filename> = restores deleted file 

git remote add origin <repo-url>

git remote set-url origin https://PAT@github.com/repo-url

git push origin master ==>>> new branch master will be created in github as the default branch in github is main
